Everything you need, nothing you don't

A focused set of features designed for practical AI-assisted coding.

Element Selection

Click any element in your running app. OpenMagic captures the DOM structure, styles, and context so the LLM knows exactly what to modify.

Smart Grounding

Automatically collects file contents, network requests, and console output. The LLM sees your actual API responses, errors, and app state.

Diff Preview

Review proposed changes as a unified diff before applying. See exactly what the AI wants to modify, line by line, then approve or reject.

Undo

Every file modification is tracked. One click to revert any change. No more fear of AI breaking your code -- roll back instantly.

Fuzzy File Matching

The AI references files by name; OpenMagic finds the right file in your project tree using fuzzy matching. No exact paths needed.

Auto-Retry

Rate limits and transient failures are handled automatically with exponential backoff. Your workflow stays uninterrupted.

Network Profiling

Captures fetch/XHR requests, response bodies, status codes, and timing. Include this context so the AI understands your API layer.

Image Attachments

Attach screenshots or design mockups directly in chat. Vision-capable models use them to understand layout and match your intent.

Keyboard Shortcuts & Session Persistence

Keyboard shortcuts for power users. Chat history and settings persist across page reloads and browser sessions automatically.

Markdown Chat

Full markdown rendering in the chat panel with syntax-highlighted code blocks. Read AI responses like documentation, not raw text.

How it works

A single-port reverse proxy that injects the toolbar. No build step, no framework plugin.

1

Start your dev server, then run OpenMagic

OpenMagic wraps your existing dev server with a reverse proxy and injects the toolbar via a script tag. Your app runs unmodified. Works with React, Vue, Svelte, Angular, Next.js, Nuxt, Astro, plain HTML -- anything with a dev server.

$ npx openmagic@latest
2

Pick a provider and paste your API key

Choose from 14 pre-configured providers and 70+ models. Select from the toolbar dropdown, paste your key once. It is stored locally in ~/.openmagic/config.json -- never sent anywhere except the provider's API.

3

Select elements, describe changes, review diffs

Click any element to ground the AI in your actual UI. Describe what you want in plain language. The AI proposes file changes as a diff -- review, approve, and your dev server hot-reloads the result instantly.
